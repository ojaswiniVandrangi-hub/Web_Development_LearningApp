üî§ HTML Elements and Attributes

Elements are the fundamental building blocks of a webpage. They define the structure, layout, and content of a website. Attributes provide extra information or control the behavior of those elements.

An HTML element consists of a start tag, content, and an end tag. Elements represent different parts of a webpage such as headings, paragraphs, images, links, lists, forms, and much more. There are over 100 standard HTML elements available.

All HTML elements can have attributes. Just like people have specifications such as height, color, or profession, HTML elements can be described or modified using attributes.

Attributes provide additional details about elements. They are always included in the opening tag and usually follow a name="value" pattern.

Real-life Example: Think of a car as an element ‚Äî its color, brand, and model are like attributes that describe the car in more detail.

Example of an element with attributes:
<a href="https://google.com">Visit Google</a>

In the above example, href="https://google.com" is the attribute, where href is the name of the attribute and https://google.com is the value.

üß© Nesting Elements

Nesting means placing elements inside other elements to build a structured web page, just like boxes inside boxes or containers inside containers.

The most basic structure of any HTML page uses nesting. Here's a perfect example: 


<html>
    <head>
        <title>My Webpage</title>
    </head>

    <body>
        <h1>Welcome to my site</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

In the above example:

  • The <html> element is the parent that wraps everything.
  • Inside <html>, we have two nested elements ‚Äî <head> and <body>.
  • Further inside <head>, the <title> is nested.
  • Inside <body>, we nest other elements like <h1> and <p>.

Real-life Example: Think of a tiffin box. The large box holds smaller containers inside it ‚Äî rice in one, curry in another. Similarly, HTML elements sit inside parent elements to stay organized.

Nesting helps maintain structure, hierarchy, and clarity in your webpage. Incorrect nesting can break your layout!

üî¢ Superscript and Subscript

Superscript: Text that appears slightly above the normal line, often used for exponents, mathematical formulas, or footnotes.

Subscript: Text that appears slightly below the normal line, commonly used in chemical formulas or technical notations.

Real-life Examples:

  • Water formula ‚Üí H2O
  • Carbon dioxide ‚Üí CO2
  • Mathematical exponent ‚Üí 52 = 25
  • Chemical expression ‚Üí CH4
  • Footnotes ‚Üí Term*

Example HTML Code:


H<sub>2</sub>O ‚Üí H2O

CO<sub>2</sub> ‚Üí CO2

5<sup>2</sup> = 25

X<sup>n</sup> = Power of n

üìè Inline vs Block Elements

Inline: Flows with text (e.g., <span>, <a>).
Block: Takes full width, starts on new line (e.g., <div>, <p>).

Real-life Example: Inline = words in a sentence, Block = a paragraph or box below.
